Endpoint (конечная точка) - это само обращение к маршруту отдельным HTTP методом. Эндпоинты выполняют конкретную задачу, принимают параметры и возвращают данные клиенту.
Мврштур (Route) - это "имя" которое отсылает роботу API к определенным эндпоинтам. Если упростить, то марштур это URL к которому можно обратится разными HTTP методами. Марштур может иметь несколько эндпоинтов.
API работают с помощью запросов и ответов. Место, куда API высылает запрос и где находится ресурс - называют эндпоинт.
