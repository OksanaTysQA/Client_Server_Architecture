Браузер != ядро браузера.
Браузер - объеденяет периферию и предоставляет функционал позволяющий манипулировать движком, его поведением, предоставляет дополнительные сервисы.
Браузерный движок выполняет всю "грязную" работу: загрузка, обработка, отрисовка данных и все расчеты.
Ядро браузера можно разделить на 2 части:
1. Движок рендеринга: он отвечает за получение содержимого веб-страницы (HTML, XML, изображения и т. д.), организацию информации (добавление CSS и т. д.), вычесление режима отображения веб-страницы, а затем вывод на монитор или принтер.
2. Движок JS: анализирует язык JS и выполняет язык JS для достижения динамического эффекта веб-страницы.

Общие ядра браузера можно разделить на типы:
1. Trident (IE ядро) также MSHTML.
Представители: Internet Explorer, Avant, Tencent.
Ответвление Trident - ядро EdgeHTML - это движок от компании Microsoft для ее браузера Microsoft Edge.
2. Gecko (ядро Firefox) - открытый движок проекта Mozilla, используется в большом числе программ, осованных на коде Mozilla.
3. Webkit (Safari) - движок для браузера Apple Safari, включенного в операционную систему MacOSx и браузера Chrome (до 2013 года). Типичные браузеры: Maxhthon, Browser 3, Apple Safari, Symbian, Android).
4. Chromium (Blink - является ответвлением WebKit. Браузеры: Google Chrome с 28 версии, Microsoft Edge с 79 версии, Opera с 15 версии и Vivald. Свободный движок для отображения веб-страниц, разработанный Google Inc. на основе кода WebCore из WebKit для браузера Chromium.
5. Presto (Opera) - браузерный движок разработанный Opera Software для браузера Opera. В 2013 году Opera отключилась от Presto в пользу Blink.
