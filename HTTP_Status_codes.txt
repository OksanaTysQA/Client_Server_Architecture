Код состояния HTTP (HTTP Status Code) - часть первой строки ответа сервера при запросах по протоколу HTTP. Целое трехразрядное десятичное число. Первая цифра указывает на класс состояния.
Клиент узнает по коду ответа о результатах своего запроса и определяет какие действия ему предпринимать дальше.
1xx Informational (информационные)
2xx Success (успешно)
3xx Redirection (перенаправление)
4xx Client Error (ошибка клиента)
5xx Server Error (ошибка сервера)
